### remote debug setting
- 모바일 기기 USB연결
- 모바일 개발자 모드 설정
- adb 설치
    - https://developer.android.com/studio/releases/platform-tools
- 설치 후 abd 실행하도록 환경 변수 설정
- cmd에서 설치 확인
    - adb devices
- chrome 브라우저 확인
    - chrome://inspect/#devices
