### pseudo elements
- Element의 특정 상황에 따라 정보를 표기하고 싶을 때 사용하는 element Tag
    - ex) Alarm Badge, ToolTip
- ::before
- ::after
- ::first-letter
- ::first-line
- ::selection

### attr()
- pseudo elements에서 target element의 속성값을 가져다가 쓰고 싶을 경우 사용하는 CSS

### pseudo classes
- :active
- :checked
- :disabled 
etc..
